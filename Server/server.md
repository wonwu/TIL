## Chapter5. 메일과 인터넷
### Proxy 서버
- proxy는 '대리'를 의미 
- 즉 클라이언트의 인터넷 통신을 대행
- 클라이언트 입장에서 Proxy는 인터넷 통신을 대행 + 효율화함 + 방화벽 역할
- Proxy 서버는 내부 네트워크와 인터넷 사이에서 엑세스를 중계함
